#Tienda GitHub


La aplicación permite administrar los productos de una tienda.

## Tabla de contenido

|Número|Contenido|Descripción|
|---|---|---|
|1|Agregar cafe|se agrega nombre, precio, descripción, nivel de tostado, disponibilidad|
|2|Editar la información del café|Permite editar la cantidad y la disponibilidad
|3|Eliminar|Permite eliminar un item de café existente en su totalidad
## Tecnología utilizada

>**Python**


## Uso

- La aplicación se ejecuta mediante un menú interactivo por consola. 

- El programa se mantine en ejecución hasta que el usuario sleccione la opción Salir.

## Instrucciones de ejecución

### Menu Principal
Será solo de presionar números que aparecen en el menú de acuerdo a su descripción. Por ejemplo el 1 "Permite agregar un item", mientras que el "4" nos permite salir del programa. 


#### Comando de ejecución del programa
``` 
main.py
```

> [!NOTE] 
>Los menú son interactivos y requieren que siga las instrucciones


## Estructura del Proyecto
- El aplicativo esta desarrolado en Python
- El aplicativo se ejecuta desde consola
- Dentro del programa esta diseñado con funciones, condicionales, ciclos y diccionarios para la persistencia
- Contiene un menú interactivo
- Coffee. json contiene los datos almacenados por medio del input
- menu.py contiene el menú visual y el menú que ejecuta los comandos del usuario por medio de el ingreso de números.
- funciones.py contiene las funciones que se realizaron para ejecutar cada uno de los comandos indicados en menú
- main.py contiene la línea de código que ejecuta el programa

> [!NOTE]
> Contiene permanencia de datos, para almacenarlos y que sean re-utilizables


## Funcionalidades 
- El aplicativo permite registrar un nuevo producto en el inventario solicitando: nombre, precio, cantidad disponible, descripción, nivel de tostado y disponibilidad.

- El aplicativo permite modificar la cantidad y la disponibilidad de un producto existente (el usuario debe ingresar el nombre del product y la nueva cantidad). 

- El aplicativo permite eliminar un producto existente por su nombre.

- El aplicativo permite salir de la ejecución del programa.
